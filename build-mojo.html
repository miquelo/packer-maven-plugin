packer:build

Full name:

io.github.miquelo.maven.plugin.packer:packer-maven-plugin:0.5.1:build

Description:

Run a Packer build command.

Attributes:

  • Requires a Maven project to be executed.

Optional Parameters

Name Type Since Description
<changesNeeded> boolean - Whether must be some change for this build in order to be executed.
Default value is: true.
<except> String - Comma-separated builder names that must be ignored.
<force> boolean - Whether build must be executed even output directory is created.
Default value is: false.
<inputDirectory> File - Input directory.
Default value is: ${project.build.directory}/packer/input.
<invalidateOnFailure> boolean - Whether working files must be invalidated when build fails.
Default value is: true.
<only> String - Comma-separated builder names that must be taken into account. Empty for all.
<skip> boolean - Whether this execution must be skipped.
Default value is: false.
<templatePath> String - Template path relative to input directory.
Default value is: template.json.
<varFiles> Set - Variable files for template. Relative to source directory.
<vars> Properties - Variables for template.
